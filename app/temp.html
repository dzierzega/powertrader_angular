Symptoms

Create a New MS Lync account.

Exception: Cole Valley Service Desk agents. 

Cause

New Starter

Checklist

The Lync Sever 2013 console can be accessed at: https://lyncpool01.scc.com/cscp/

 

Resolution

Lync accounts are to be created for all users except Cole Valley Service Desk agents.

The Lync Sever 2013 console can be accessed at: https://lyncpool01.scc.com/cscp/

Once logged in you will be presented with the below screen:

  • Select "Users".

  • Click on the "Enable users" button.

  • Press "Add…".

  • Type in the name of the user and press "Find".
  • Select the user you need to create the account for and press "OK".

  • On the next screen press the arrow on the dropdown box for "Assign users to a pool".

  • Select "lyncpool01.scc.com".
  • All the other settings should be left on their default values.
  • Press "Enable" at the top.

The MS Lync account is now created.
